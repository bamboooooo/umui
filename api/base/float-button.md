# FloatButton组件
> 漂浮按钮组件
## 一、功能说明
### 固定在页面的某个位置，不会随页面滚动而滚动。

---

## 二、属性说明
> 组件属性通过props进行传递。

参数 | 说明 | 类型 | 默认值
---|---|---|---|---
className | 样式类名 | string |无
id|dom元素id|string|无
img|按钮的背景图片|string|无
text|按钮的文案|string|无
onClick|绑定click事件|function|无


---

## 三、方法说明
### 1.show()
> 显示漂浮按钮组件

#### 参数
- v: bool，true为开，false为关

#### 返回值
- void


### 2.hide()
> 隐藏漂浮按钮组件

#### 返回值
- void
