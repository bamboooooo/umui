# Tag组件
> 角标用于提示，提醒等
## 一、功能说明
### 1.有新消息或内容有更新时，提醒用户查看。

---

## 二、属性说明
> 组件属性通过props进行传递。
> 组件内容通过children进行传递。

参数 | 说明 | 类型 | 默认值
---|---|---|---|---
className | 样式类名 | string |无
displayText | 是否显示文本 | bool | true
id|dom元素id|string|无

---

## 三、方法说明
### 1.setValue(t)
> 设置组件的文本内容

#### 参数
- t: string类型，设置的值

#### 返回值
- void

### 2.show()
> 设置组件显示

#### 参数
- 无

#### 返回值
- void

### 3.hide()
> 设置组件隐藏

#### 参数
- 无

#### 返回值
- void

` <Tag>new</Tag> 最终会被渲染为 <i>new</i>`
