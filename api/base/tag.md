# Tag组件
> 角标用于提示，提醒等
## 一、功能说明
### 1.有新消息或内容有更新时，提醒用户查看。

---

## 二、属性说明
> 组件属性通过props进行传递。
> 组件内容通过children进行传递。

参数 | 说明 | 类型 | 默认值
---|---|---|---|---
className | 样式类名 | string |无
id|dom元素id|string|无
value|显示的文本|string|无

---

## 三、方法说明
### 1.setValue(v)
> 设置组件的文本内容

#### 参数
- v: string类型，设置的值

#### 返回值
- void

### 2.getValue()
> 获取组件的文本内容

#### 返回值
- void

### 3.show()
> 设置组件显示

#### 参数
- 无

#### 返回值
- void

### 4.hide()
> 设置组件隐藏

#### 参数
- 无

#### 返回值
- void
